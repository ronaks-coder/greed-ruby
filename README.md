# Greed (Ruby)

A command-line implementation of the Greed dice game in Ruby.

## Requirements

- Ruby 2.7+ (works with modern Ruby 3.x)
- Bundler (recommended)

## Setup

```bash
git clone <repo>
cd greed-ruby
bundle install
bundle exec rake
```

## Run

```bash
ruby bin/greed
# or
bundle exec ruby bin/greed
```

## Tests

```bash
bundle exec rake
```

## Project Structure

- `lib/greed` — game implementation files
- `bin/greed` — executable
- `spec/` — RSpec tests

Enjoy!
